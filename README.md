# Banking Administration System

A fully functional banking system built in Java for INFO1113 
Object-Oriented Programming at the University of Sydney (2025).
Achieved 19.5/20.

## What it does
- Create and manage bank accounts
- Process and track transactions with hash verification
- Archive and recover ledgers
- Statistical commands (MAX, MIN, MEAN, TOTAL)

## Concepts applied
OOP · Inheritance · Interfaces · Exception handling · Collections

## How to run
javac Banker.java && java Banker
